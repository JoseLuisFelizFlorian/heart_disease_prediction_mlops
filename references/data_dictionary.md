
# Diccionario de Datos - Heart Disease Prediction

| Columna | Descripción | Valores / Unidades |
|:---|:---|:---|
| **Age** | Edad del paciente | Años |
| **Sex** | Sexo del paciente | M: Masculino, F: Femenino |
| **ChestPainType** | Tipo de dolor de pecho | TA, ATA, NAP, ASY |
| **RestingBP** | Presión arterial en reposo | mm Hg |
| **Cholesterol** | Colesterol sérico | mm/dl |
| **FastingBS** | Azúcar en sangre en ayunas | 1: > 120 mg/dl, 0: Contario |
| **RestingECG** | Resultados ECG en reposo | Normal, ST, LVH |
| **MaxHR** | Frecuencia cardíaca máxima | 60-202 |
| **ExerciseAngina** | Angina por ejercicio | Y: Sí, N: No |
| **Oldpeak** | Depresión del ST | Numérico |
| **ST_Slope** | Pendiente del segmento ST | Up, Flat, Down |
| **HeartDisease** | **TARGET** | 1: Enfermedad, 0: Normal |
